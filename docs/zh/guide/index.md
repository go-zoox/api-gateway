# 指南

欢迎使用 API Gateway 指南！本文档将帮助您快速开始使用 API Gateway 并学习其功能。

## 什么是 API Gateway？

API Gateway 是一个使用 Go 构建的简单、强大、灵活的自托管 API 网关。它提供：

- **简单配置**：基于 YAML 的配置，易于理解和维护
- **灵活路由**：支持前缀和基于正则表达式的路径匹配
- **路径重写**：高级路径重写规则，用于请求转换
- **插件系统**：可扩展的插件架构，支持自定义功能
- **健康检查**：内置网关和后端服务的健康检查
- **服务发现**：基于 DNS 的服务发现
- **身份验证**：支持多种身份验证方法
